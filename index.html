¿De que habla este proyecto?

Se busca la manera de generar electricidad limpia, sustentable mediante induccion y movimiento con la suficinete energia para cargar un celular

Los materialez que usameros para el proyecto son:

  • Motor con un bobinado de cobre reciclado de una impresora obsoleta
  • controlador de carga reciclado de un cargador portatil con la bateria muerta
  • madera reciclada para la contruccionde la caja que contiene todos los elementos que generan la electricidad

creo que todos al ver el proyecto nos preguntamos ¿como es que todo eso genera electicad?
